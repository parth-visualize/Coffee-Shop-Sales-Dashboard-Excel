## 🎬 Amazon Prime Titles Data Analysis (Python)

## 📌 Project Overview
- This project performs Exploratory Data Analysis (EDA) on the Amazon Prime Titles dataset using Python.
- The goal of the project is to analyze the available movies and TV shows on Amazon Prime and extract meaningful insights from the dataset.
- The analysis includes data cleaning, preprocessing, and visualization to better understand the content distribution.

## 🎯 Project Objectives
- Explore the dataset and understand its structure.
- Clean and preprocess the data for analysis.
- Analyze the distribution of movies vs TV shows.
- Identify trends in release years and content types.
- Visualize the data to generate meaningful insights.

## 📊 Key Analysis Performed
- Distribution of Movies vs TV Shows
- Content distribution by country
- Trends based on release year
- Analysis of genres/categories
- Ratings distribution
- Visual insights using charts and graphs

## 🗂 Dataset Information
The dataset contains information about titles available on Amazon Prime Video, including:
- Show ID
- Title
- Type (Movie or TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed In)
- Description

## 🧹 Data Cleaning Steps
- The notebook performs several preprocessing steps:
- Removing unnecessary columns
- Checking dataset structure and missing values
- Handling missing values in:
  1. Director
  2. Cast
  3. Country
- Preparing data for visualization and analysis
